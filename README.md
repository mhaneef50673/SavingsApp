# SavingsApp
Savings App

# How to Run

1. clone the repo
2. run "npm" install from the project directory
3. start your emulator
4. run "react-native run-android"

Or 
1. Install the signed apk from "/apk/Savings App.apk" in any device/emulator 
2. Launch the App
